Controls:

- ESC (to close the window).

- W (to go up).
- A (to go backwards).
- S (to go down).
- D (to go forwards).

- UP ARROW (to move up the screen).
- LEFT ARROW (to move backwards the screen).
- DOWN ARROW (to move down the screen).
- RIGHT (to move forwards the screen).
